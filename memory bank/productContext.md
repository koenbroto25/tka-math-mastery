# Product Context: TKA Math Mastery

## Project Vision
To create an immersive, RPG-gamified mathematics learning platform ("The Kingdom of Arith") for SD (Elementary) and SMP (Junior High) students in Indonesia. The platform combines high-quality video content, interactive quizzes, and guided drills into a "Hero's Journey".

## Core Product Features

### 1. The Learning Journey (Student Experience)
- **Interactive Video (The Vision)**: 
  - Students watch animated math videos.
  - **Feature**: Videos strictly pause at key moments to ask questions.
  - **Constraint**: Students *must* answer correctly to proceed. This ensures active engagement and comprehension, not just passive watching.
  - **Auto-Resume**: Seamless playback continuation after a correct answer.
- **Training Grounds (Guided Drills)**: Step-by-step problem solving where students fill in the blanks.
- **Shadow Training**: Practice mode with hints.
- **Guardian Skirmish**: Standard quizzes to test knowledge.
- **The Time Rift**: Cumulative exams to test retention of past topics.

### 2. Grandmaster Citadel (Admin Dashboard)
A centralized command center for content creators and administrators.

#### **Shadow Forge (Video Editor)** - *Major Update*
The core tool for turning passive YouTube videos into interactive learning experiences.
- **Draggable Timeline**: A visual editor below the video player allowing admins to drag-and-drop quiz markers to precise timestamps. Markers are visually distinct (red, glowing) for ease of use.
- **Real-time Sync**: Dragging a marker immediately updates the timestamp in the editor form.
- **Edit Mode**: Admins can edit existing quizzes (Time, Question, Options, Answer) without deleting and recreating them.
- **Magic Auto-Forge**: An AI-powered assistant that reads PDF materials and video scripts to automatically generate **5 perfectly timed quizzes** (2 Theory, 3 Problem Applications) and places them at the logical end of explanation scenes.

#### **Scroll Forge (Content Sync)**
- A tool to upload/sync JSON data for materials and questions directly to the database.
- Supports bulk updates for quizzes, drills, and materials.

#### **Video Automator (Grok Engine)**
- Generates detailed AI prompts to create math animation videos (Stickman style) from raw text materials.

#### **Simulation Hub (Omni Gate)**
- **God Mode**: Allows admins to simulate Student, Teacher, or Parent roles to test the user experience.
- **Teleport**: Instantly jump to any specific mission or chapter page without navigating the full map.

## Monetization & Access Control
- **Freemium Model**: Basic access is free, but premium features (like detailed analytics or advanced drills) might be gated.
- **Teacher Ecosystem**: Teachers can create classes and monitor student progress (Mentor Guild).
- **Verification**: Teachers can get "Verified" status by inviting a certain number of students/parents.

## User Personas
- **Student (Hero)**: Wants to learn math in a fun, game-like way. Motivated by XP, Gold, and unlocking new levels.
- **Teacher (Mentor)**: Wants high-quality materials and tools to track student performance.
- **Parent (Guardian)**: Wants to monitor their child's learning progress.
- **Admin (Grandmaster)**: Needs efficient tools to create, manage, and update content at scale.
