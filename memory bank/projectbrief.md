# Project Brief: TKA Math Mastery

TKA Math Mastery is an adaptive learning platform based on an interactive Ebook, specifically designed for preparing students for the Math Academic Competency Test (Tes Kompetensi Akademik - TKA) at SD (Elementary) and SMP (Junior High) levels.

## Core Goals
- Provide an engaging and interactive learning experience for Math.
- Implement an adaptive learning flow: The Sage's Briefing -> Shadow Training -> Guardian's Skirmish -> The Time Rift.
- Gamify the experience to motivate students using RPG-style mechanics (XP, Levels, Boss Battles with HP bars).
- Create a sustainable "Social Recovery" model: Use social engagement (Share/Subscribe) to recover "Hearts/Energy".
- Provide monitoring tools for parents and a comprehensive Mentor Guild (Marketplace) for Teachers.
- Create a scalable and lightweight web application using modern static site technologies and AI-driven content management.

## Core Requirements
- **Frontend**: Lightweight, fast, and offline-capable (PWA).
- **Backend**: Scalable database and authentication using Supabase.
- **Gamification**: XP system, levels, and interactive combat-style exercises.
- **Multi-role Access**: Distinct dashboards and features for Students, Parents, Teachers, and Admins.
- **Content**: Structured curriculum for SD and SMP math with video explanations and rapid calculation techniques ("Jurus Cepat").

## Scope
- Interactive learning materials with "Sage's Briefing" mascot UI and video locking.
- Dynamic exercise engine supporting TKA 2025 standards (PG, PGK MCMA, PGK Kategori).
- Turn-based RPG "Boss Battle" UI with HP bars and combat animations.
- Difficulty levels (L1/L2/L3) mapped to TKA cognitive levels.
- Energy (Hearts) system with social/ad-based recovery.
- Teacher Marketplace (Mentor Guild) with verified registration, booking, and Midtrans-integrated escrow payments.
- Admin Dashboard (CMS) featuring "AI JSON Sync" for bulk-uploading curriculum content and exercises.
- Multi-tier Monetization: Freemium access, premium "Sage Scrolls", native native ads for tutors, and B2B school licenses.
- Parent-Student account pairing and "Share to Unlock" premium reports.
